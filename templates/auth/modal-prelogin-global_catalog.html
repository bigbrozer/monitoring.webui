New authentication method

Please note that we now use AD Global Catalog in order to log into Monitoring tools using now your normal user account. This is valable for all our tools!

What does it mean ?

Previously, we used the administrative account (the ones starting with 0...), this is not more the case. You must now use your normal user account to login.

So, how do I login now ?

Log in using your normal user account, but account are of the form login@corp in AD. Please use the same form.

Note: The domain should always be corp for normal user accounts.
Example

My user account is besancon so I will log using besancon@corp. This is not related to the CORP administrative account !
